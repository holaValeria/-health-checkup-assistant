# 体检问卷智能推荐决策树逻辑（优化版）

## 一、基础体检项目（所有受检者）

- 身高、体重、腰围测量与BMI计算
- 血压测量
- 血常规、尿常规
- 肝肾功能（ALT、AST、GGT、肌酐、尿素氮、**尿酸**）
- 血脂四项（总胆固醇、甘油三酯、LDL-C、HDL-C）
- 空腹血糖
- **选择性胸部X线片**（根据风险因素决定，非常规）

## 二、年龄相关推荐（分层递进，优化版）

### 18-29岁
- 基础体检项目
- 视力检查

### 30-39岁（在18-29岁基础上增加）
- 腹部超声（肝胆胰脾）
- 心电图
- 甲状腺功能检查适用人群：
  * 有甲状腺疾病家族史
  * 有甲状腺相关症状
  * **女性计划妊娠者**
  * **BMI≥24者加甲状腺抗体检测**
- 乙肝两对半（未接种疫苗者）

### 40-49岁（在30-39岁基础上增加）
- 胃镜检查：
  * **胃癌高风险人群**：每2-3年（幽门螺杆菌阳性/萎缩性胃炎/胃癌家族史）
  * **普通人群**：有症状者每5年
- 男性：前列腺特异性抗原(PSA)（**告知假阳性风险和过度诊断问题**）
- 女性：
  * 乳腺超声+宫颈TCT（HPV未接种者加HPV检测）
  * **乳腺密度评估，致密型乳腺需补充乳腺MRI**
- 骨密度筛查（女性更年期前后）
- 眼底检查（有高血压、糖尿病者）

### 50-59岁（在40-49岁基础上增加）
- 结肠镜检查：
  * **首次筛查：50岁**
  * **高风险者**（家族史/炎症性肠病）：每5年
  * **普通人群**：每10年
  * **拒绝肠镜者**：每年FIT+每3年乙状结肠镜
- 男性：前列腺超声
- 女性：乳腺钼靶（每1-2年）
- 肺部低剂量CT（仅限高危人群：吸烟≥20包年或有职业暴露）
- 颈动脉超声（有动脉粥样硬化风险因素者）
- 糖化血红蛋白（**仅限糖尿病前期患者**：空腹血糖5.6-6.9 mmol/L或BMI≥25）

### 60岁及以上（在50-59岁基础上增加）
- 脑部MRI/CT：**非常规筛查**，仅推荐给：
  * **认知功能评估异常**
  * **有卒中史**
  * **高血压III级合并腔隙性脑梗风险**
- 心脏超声（有心血管疾病风险因素者）
- 听力检查
- 认知功能评估
- 肺功能检查（吸烟者或有呼吸系统症状）
- 老年人全面评估（跌倒风险等）

## 三、风险因素分值系统与检查推荐（优化版）

### 心血管疾病风险评分
| 风险因素 | 分值 |
|---------|------|
| 年龄≥50岁 | +1分 |
| 男性或绝经后女性 | +1分 |
| 吸烟 | +2分 |
| 高血压 | +2分 |
| 糖尿病 | +2分 |
| 血脂异常 | +1分 |
| BMI≥28 | +1分 |
| **腰围超标（男≥90cm/女≥85cm）** | **+2分** |
| 家族心血管病史 | +2分 |
| 缺乏运动 | +1分 |
| **高敏C反应蛋白(hs-CRP)升高史** | **+1分** |
| **慢性肾病（eGFR<60）** | **+3分** |

**风险评分与推荐检查**:
- 0-2分：基础心血管检查（血压、血脂、血糖、心电图）
- 3-5分：增加心脏超声、颈动脉超声
- 6-8分：增加冠状动脉CT、运动负荷实验
- ≥9分：心脏内科专科评估、必要时冠脉造影
- **高血压患者**：补充**动态血压监测**（识别隐匿性高血压）

### 肿瘤风险评分
| 风险因素 | 分值 |
|---------|------|
| 年龄≥50岁 | +1分 |
| 吸烟≥10年 | +3分 (肺癌) |
| 长期饮酒 | +2分 (消化道癌) |
| 家族癌症史 | +3分 (对应癌种) |
| 职业暴露 | +2分 (相关癌种) |
| 慢性感染（如HBV） | +2分 (肝癌) |
| 肥胖(BMI≥28) | +1分 |
| 相关症状(血便、进行性吞咽困难等) | +3分 |

**肺癌筛查触发条件**:
- 吸烟累计≥20包年 AND (年龄≥50岁 OR 职业暴露 OR 肺癌家族史)
- 推荐项目：低剂量螺旋CT

**消化道肿瘤筛查触发条件**:
- 年龄≥50岁 OR 有症状(血便、黑便、进行性消瘦) OR 家族史
- 推荐项目：胃镜、结肠镜

**乳腺癌筛查个性化方案**:
- 40-44岁：乳腺超声，有风险因素者每年一次
- 45-54岁：乳腺超声+钼靶，每1-2年一次
- 55岁以上：钼靶，每2年一次
- 乳腺癌家族史：从40岁或比最年轻患病亲属提前10年开始筛查
- **乳腺密度评估：致密型乳腺需补充乳腺MRI**

**子宫颈癌筛查个性化方案**:
- 21-29岁：TCT，每3年一次
- 30-65岁：TCT+HPV，每5年一次（已接种HPV疫苗者可延长间隔）
- HPV阳性：需增加阴道镜检查

**前列腺癌筛查个性化方案**:
- 需进行**共同决策，告知假阳性风险和过度诊断问题**
- 50-70岁男性：PSA检测，每2-4年一次
- 有家族史：从45岁开始筛查
- PSA>4ng/ml：增加前列腺超声和必要时活检

## 四、特殊人群跳转和推荐逻辑（优化版）

### 女性避孕方式相关检查
- 口服避孕药：肝功能、血脂、血压、血栓风险评估、**凝血因子V Leiden突变筛查**（血栓高风险人群）
- 宫内节育器：宫腔超声、HPV检测

### 男性前列腺筛查策略
- 40-54岁，无症状/无家族史：基本体检
- 40-54岁，有症状/有家族史：PSA+直肠指检
- ≥55岁：PSA+直肠指检，异常者增加前列腺超声
- **BMI≥25且合并疲劳症状**：增加**性激素六项**（包括睾酮水平）

### 职业暴露特殊检查
- 粉尘暴露：肺功能、胸部X线片或高分辨CT
- 化学物质：肝肾功能、血常规、尿液分析、**根据具体物质细化检查**（如苯暴露→骨髓穿刺）
- 放射性暴露：血常规、染色体畸变分析
- 噪音暴露：听力检查、**前庭功能检查**（眩晕风险评估）

### 慢性病随访特殊检查
- 高血压：尿微量白蛋白、眼底检查、心电图、超声心动图、**动态血压监测**
- 糖尿病：糖化血红蛋白、尿微量白蛋白、眼底检查、足部检查、**心脏自主神经功能检测**（预防无痛性心梗）
- 慢性肝病：肝纤维化指标、肝脏超声、甲胎蛋白

## 五、症状导向检查推荐（优化版）

### 症状触发特定检查逻辑
- **持续咳嗽>2周**：
  1. 先做胸片+肺功能
  2. 若阴性但症状持续→低剂量CT
- 消化道症状：胃镜、结肠镜、腹部超声
- 尿路症状：尿常规、尿培养、泌尿系统超声
- **不明原因出血**：
  * 凝血功能
  * **血管性血友病因子检测**
  * **血小板持续减少时考虑骨髓穿刺**
- **疲劳+体重下降**：
  * 优先排查：**甲状腺功能亢进、结核菌素试验**
  * 若初筛正常：全面血液检查、肿瘤标志物筛查、胸腹盆CT

### 症状持续时间与严重程度评估
- <2周轻微症状：随访观察
- 2周-3个月持续症状：针对性检查
- >3个月症状：全面评估+专科会诊

## 六、示例：完整推荐逻辑流程（优化版）

### 示例1：45岁男性
- 基本信息：吸烟20年，父亲有心脏病史，BMI=26
- 问卷回答：有持续性咳嗽3个月，无其他症状

**推荐套餐**：
1. 基础体检套餐（**加测尿酸**）
2. 心血管评估套餐（心电图、心脏超声、颈动脉超声）
3. 呼吸系统评估：
   * 第一阶段：胸部X线片+肺功能检查
   * 第二阶段：如果症状持续且前期检查阴性，进行低剂量螺旋CT
4. 前列腺筛查（PSA）**（告知假阳性风险和过度诊断问题）**
5. 结肠镜检查（如近5年未做）
6. **性激素六项**（BMI>25且有疲劳症状）

### 示例2：35岁女性
- 基本信息：从不吸烟，母亲有乳腺癌史，BMI=21
- 问卷回答：月经不规律，**计划一年内妊娠**，无其他症状

**推荐套餐**：
1. 基础体检套餐
2. 女性健康套餐：
   * 乳腺超声（**评估乳腺密度，若致密考虑MRI**）
   * 妇科彩超
   * 宫颈TCT+HPV
3. 内分泌功能检查（性激素六项、**甲状腺功能+抗体**）
4. **叶酸、维生素D水平**（计划妊娠者）

### 示例3：62岁女性
- 基本信息：从不吸烟，有糖尿病10年史，BMI=29
- 问卷回答：有轻度尿频、视力下降

  * **血小板持续减少时考虑骨髓穿刺**
- **疲劳+体重下降**：
  * 优先排查：**甲状腺功能亢进、结核菌素试验**
  * 若初筛正常：全面血液检查、肿瘤标志物筛查、胸腹盆CT

### 症状持续时间与严重程度评估
- <2周轻微症状：随访观察
- 2周-3个月持续症状：针对性检查
- >3个月症状：全面评估+专科会诊

问卷系统将根据用户回答不断调整后续问题，例如：
- 如选择"有高血压"→自动展开高血压相关追问，并推荐**动态血压监测**
- 如选择"有持续咳嗽"→自动展开肺部症状详细评估，**采用分步策略检查**
- 如性别为"女性"且"计划妊娠"→自动增加**甲状腺功能、叶酸、维生素D检测**
- 如选择"口服避孕药"→评估血栓风险，高风险者考虑**凝血因子V Leiden突变筛查**

这种动态决策树结构确保问卷既全面又高效，避免不必要的检查，同时针对个体风险因素提供最科学、最优化的体检方案。

一、问卷基本结构
1. 基本信息收集部分

年龄分组：使用单选题，分为5个年龄段（18-29岁、30-39岁、40-49岁、50-59岁、60岁及以上）
性别：单选题（男性/女性）
基础身体数据：数值填空题（身高、体重、腰围）

设置自动计算BMI的公式：BMI = 体重(kg) / (身高(m))²


家族史：多选题，包含心血管疾病、各类肿瘤（胃癌、结肠癌等）和甲状腺疾病

2. 生活方式风险评估

吸烟史：级联问题

首先询问吸烟状态（从不/曾经/目前）
如选择"曾经"或"目前"，追问吸烟年数和日均支数，自动计算包年


饮酒习惯：类似吸烟的级联结构
运动习惯：单选题（规律/偶尔/缺乏）
职业暴露：多选题（粉尘/化学物质/放射线/噪音）

3. 现有健康状况

慢性疾病史：多选题，选择后展开细节问题

例如选择"高血压"，则追问诊断时间和控制情况


最近体检数据：数值填空题或范围选择题
用药情况：多选题，特殊药物（如女性避孕药）追加相关问题

二、逻辑跳转设置
1. 基于年龄的跳转
根据选择的年龄段，自动确定基础体检项目清单：

18-29岁：仅推荐基础项目和视力检查
30-39岁：增加腹部超声和心电图
40-49岁：增加性别相关筛查
50-59岁：增加结肠镜等
60岁及以上：增加老年评估项目

2. 基于性别的跳转

男性：显示前列腺相关问题，设置40岁和55岁两个分支点
女性：显示妇科相关问题，包括月经情况、计划妊娠、避孕方式

如选择"计划妊娠"：自动增加甲状腺功能、叶酸、维生素D检测
如选择"口服避孕药"：增加血栓风险评估问题



3. 基于症状的跳转

持续咳嗽：追问持续时间，根据回答（<2周/2周-3个月/>3个月）提供不同检查推荐
消化系统症状：根据症状严重性推荐相应检查
不明原因出血：触发凝血功能检查问题
疲劳+体重下降：触发甲状腺和结核相关问题

三、后台计分系统设置
1. 心血管疾病风险评分
设置隐藏计算字段，根据回答自动计算分数：

年龄≥50岁：+1分
男性或绝经后女性：+1分
吸烟：+2分
高血压：+2分
糖尿病：+2分
血脂异常：+1分
BMI≥28：+1分
腰围超标：+2分
家族心血管病史：+2分
缺乏运动：+1分
高敏C反应蛋白升高史：+1分
慢性肾病：+3分

根据总分，设置条件显示不同级别的心血管检查推荐：

0-2分：基础心血管检查
3-5分：增加心脏超声、颈动脉超声
6-8分：增加冠状动脉CT、运动负荷实验
≥9分：心脏内科专科评估

2. 肿瘤风险评分
类似心血管风险评分，设置隐藏计算字段，针对不同肿瘤类型：

肺癌筛查触发条件：吸烟≥20包年 AND (年龄≥50岁 OR 职业暴露 OR 家族史)
消化道肿瘤筛查：年龄≥50岁 OR 相关症状 OR 家族史

四、最终推荐系统
基于所有收集的信息和计算的风险评分，生成个性化体检推荐套餐：

基础体检套餐：所有人群基础项目+年龄相关项目
心血管评估套餐：根据心血管风险评分定制
肿瘤筛查套餐：根据肿瘤风险评估和触发条件定制
慢性病随访套餐：有慢性病史者推荐相应监测项目
症状导向检查套餐：根据症状评估结果定制
特殊人群套餐：针对特殊情况（如职业暴露、避孕方式等）定制
